工作紀錄簿撰寫說明 
一、概述 
本單位執行科技研究發展專案計畫，為提供主管機關經濟部於進行 期中、期末查訪同時，將進行工作紀錄簿之查核，俾瞭解參與科技 專案同仁之平時工作情形，以確實掌握專案執行狀況。 
二、目的 
記錄員工之研究、實驗、會議摘要、個人心得、發現及創意等，俾 保障研究成果以為未來可能之智財權糾紛時之佐證。 
三、依據 
經濟部「經濟部協助產業創新活動補助獎勵及輔導辦法」辦理。 
四、適用對象
執行科技專案之所有人員。
五、適用時機 
凡投入科技專案執行人員自投入之日起開始撰寫，且研究紀錄簿原 則由上一層主管簽名認證。 
六、撰寫方式
請使用可長久保留筆跡之書寫工具書寫，記載內容無一定格式，以 清晰易瞭解為原則。若以黏貼方式紀錄，須於黏貼處親自簽名蓋章，紀錄簿不得撕頁且中間不可留空白頁。記錄錯誤時請用筆刪去 即可，不得割除、貼掉或以修正液塗掉。 
七、撰寫內容 
請記錄各項研究工作、工程技術、設計結構與分析等之數據、資料 及其改變，或重要會議、信件、談話，個人研究心得、發現及創 意，相關行政業務紀錄等。非科技專案之內容請勿載入，以免與科 技專案之成果產生智財權上的糾紛。
八、見證時機 
定期呈主管見證，若遇重大發現、發明、心得或創意等，應隨即送 請見證。
九、保證 
應善盡紀錄簿保管之責，非經上層主管同意不得展示、影印，或對 外揭露記載內容，如不再參與本專案應將紀錄簿繳還計畫主持人。

```dart
Future<bool> _requestBlePermissions({required bool showDeniedDialog}) async {
  final updatedStatuses = <Permission, PermissionStatus>{};

  for (final entry in _blePermissions.entries) {
    final status = await entry.key.request();  // 觸發系統權限對話框
```

#### 2. 配對流程：手機掃描 → 配對碼驗證 → 建立配對記錄
- **對應模組**：藍牙設備掃描與連線管理

- **模組說明**：
  本模組實現使用者友善的藍牙設備掃描、發現與配對流程。應用透過掃描環境中的藍牙廣播信號，向使用者呈現可用設備列表，使用者選擇並確認配對後，系統建立持久化的配對記錄供日後快速重連。此流程包含安全性驗證機制（如可選的配對碼驗證）與狀態管理：
  - **功能**：
  (1)啟動藍牙掃描並發現周圍設備
  (2)展示設備清單供使用者選擇
  (3)執行配對碼驗證或通過 BLE Bonding 機制建立信任關係

```dart
Future<Map<String, String>> finishRoundLogging() async {
  final results = <String, String>{};
  
  // (8a) 遍歷所有活躍日誌，刷新並關閉檔案串流
  for (final entry in _activeLogs.entries) {
    // 刷新暫存在隊列中的待寫入樣本
    _flushPendingSamples(entry.value, force: true);
    
    // 刷新 IOSink，確保所有資料寫入檔案系統
    await entry.value.sink.flush();
    
    // 關閉檔案串流

    results[alias] = entry.value.filePath;

  // (8c) 清空 _activeLogs 表，重置錄製狀態
  _activeLogs.clear();
  _roundStartClockUs = null;
```

#### 4. 資料傳輸：IMU 資料 → 序列化 → 藍芽傳輸 → 手機接收
- **對應模組**：IMU 資料日誌記錄與 CSV 寫入

- **模組說明**：
  本模組實現 IMU 感測資料的即時接收、序列化與持久化寫入。當藍牙設備透過 GATT 通知（Notification）推送感測樣本時，應用程式接收並將其存入記憶體隊列以待批量寫入，同時實現多感測器資料的同步機制（確保同一時刻的加速度與旋轉向量配對寫入）。本模組維護高效的資料流管線，以支援 100 Hz 採樣率下的持續資料記錄：

```dart
// 來自 imu_data_logger.dart (L257-295)：(5)(6)(7) 計算時戳、序列化、寫入
void _writeCombinedSample(
  _ActiveImuLog log,
  Map<String, dynamic>? linear,
  Map<String, dynamic>? rotation,
) {
  if (linear == null && rotation == null) {
    return;
  }
  
  // 防護：檢查 sink 是否已關閉
  if (log.isClosed) {
    debugPrint('⚠️ 嘗試寫入已關閉的 IMU 日誌串流');
    return;
  }
  
  // (5) 計算相對時戳（microseconds since start）
  final nowUs = DateTime.now().microsecondsSinceEpoch;
  _roundStartClockUs ??= nowUs;
  final wallClockElapsedUs = nowUs - _roundStartClockUs!;
  final wallClockElapsedSeconds = wallClockElapsedUs >= 0 ? wallClockElapsedUs / 1e6 : 0.0;
  final safeElapsed = wallClockElapsedSeconds.isFinite ? wallClockElapsedSeconds : 0.0;

  // (6) 序列化為 CSV 行格式
  final values = <String>[
    safeElapsed.toStringAsFixed(6),  // 時戳（秒數）
    _formatNumeric(rotation?['i']),  // QuatI
    _formatNumeric(rotation?['j']),  // QuatJ
    _formatNumeric(rotation?['k']),  // QuatK
    _formatNumeric(rotation?['w']),  // QuatW
    _formatNumeric(linear?['x']),    // AccelX (m/s²)
    _formatNumeric(linear?['y']),    // AccelY (m/s²)
    _formatNumeric(linear?['z']),    // AccelZ (m/s²)
  ];
  
  // (7) 調用 sink.writeln() 寫入 CSV 檔案
  log.sink.writeln(values.join(','));
}
```

```dart
// 來自 imu_data_logger.dart (L237-254)：(8) 批量刷新機制
void _flushPendingSamples(_ActiveImuLog log, {required bool force}) {
  if (!force) {
    // 非強制模式：只寫入已配對的樣本
    _drainSynchronizedSamples(log);
    return;
  }

  // (8a) 強制模式：force=true 時仍只保留最小長度的有效資料
  while (log.linearQueue.isNotEmpty && log.rotationQueue.isNotEmpty) {
    final linear = log.linearQueue.removeFirst();
    final rotation = log.rotationQueue.removeFirst();
    _writeCombinedSample(log, linear, rotation);
  }

  // (8b) 任何尚未配對的殘餘資料都直接清除，確保輸出長度一致
  log.linearQueue.clear();
  log.rotationQueue.clear();
}
```

```dart

Future<Map<String, String>> finishRoundLogging() async {
  final results = <String, String>{};
  
  // (9a) 遍歷所有活躍日誌
  for (final entry in _activeLogs.entries) {
    // (9b) 強制刷新暫存在隊列中的所有樣本
    _flushPendingSamples(entry.value, force: true);
    
    // (9c) 刷新 IOSink，確保所有資料寫入檔案系統

    // (9d) 關閉檔案串流

    final alias = _devices[entry.key]?.slotAlias ?? entry.key;
    results[alias] = entry.value.filePath;
  }
  
  // (9e) 清空錄製狀態
  _activeLogs.clear();
  _roundStartClockUs = null;
  
  return results;
}
```

```dart
void _handleLinearAccelerationPacket(String deviceId, List<int> value) {
  if (value.isEmpty || !mounted) return;

  Map<String, dynamic>? sample;

  // ========== 解析多個樣本 ==========

    sample = parsed;
    offset += 14;

Map<String, dynamic>? _parseThreeAxisSample(List<int> data, int offset) {
  final timestamp = _readUint32At(data, offset + 4);
  final rawX = _readInt16At(data, offset + 8);
  final rawY = _readInt16At(data, offset + 10);
  final rawZ = _readInt16At(data, offset + 12);
  
  if (timestamp == null || rawX == null || rawY == null || rawZ == null) {
    return null;
  }

  const double scale = 0.001; // 轉換為公尺每秒平方
  return {
    'id': data[offset],
    'seq': data[offset + 1],
    'status': data[offset + 2],
    'timestampUs': timestamp,
    'x': rawX * scale,
    'y': rawY * scale,
    'z': rawZ * scale,
    'rawX': rawX,
    'rawY': rawY,
    'rawZ': rawZ,
  };
}
```

```dart
  bool readInitialValue = true,
```

```dart
Future<void> _triggerMotorPulseForSlot(_ImuSlotType slot) async {
  final characteristic = _motorCharacteristicForSlot(slot);
  if (characteristic == null) {
    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(content: Text('${slot.displayLabel} 尚未取得震動控制特徵，請確認裝置已連線。')),
    );
    return;
  }

  if (_isMotorBusyForSlot(slot)) {
    _logBle('${slot.displayLabel} 震動命令仍在執行中，略過重複請求');
    return;
  }

  _setMotorBusyForSlot(slot, true);

  final useWithoutResponse = !characteristic.properties.write && characteristic.properties.writeWithoutResponse;

  try {
    await _bleOperationQueue.enqueue(() async {
      await characteristic.write([1], withoutResponse: useWithoutResponse); // 啟動
      await Future.delayed(const Duration(milliseconds: 500));
      await characteristic.write([0], withoutResponse: useWithoutResponse); // 停止
    }, spacing: const Duration(milliseconds: 320));
    _logBle('${slot.displayLabel} 震動命令已發送');
  } catch (error, stackTrace) {
    _logBle('震動命令失敗：${characteristic.uuid.str}，原因：$error', error: error, stackTrace: stackTrace);
    if (mounted) {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(content: Text('${slot.displayLabel} 震動失敗：$error')),
      );
    }
  } finally {
    _setMotorBusyForSlot(slot, false);
  }
}
```

```dart
Future<void> _readAndAssignString(
  BluetoothCharacteristic? characteristic,
  void Function(String) assignCallback,
) async {
  if (characteristic == null) return;
  try {
    final value = await characteristic.read();
    if (value.isEmpty) {
      _logBle('讀取特徵值為空：${characteristic.uuid.str}');
      return;
    }
    final text = _decodeUtf8String(value);
    if (text.isNotEmpty) {
      assignCallback(text);
    }
    _logBle('成功讀取字串特徵：${characteristic.uuid.str}，內容：$text');
  } catch (error, stackTrace) {
    _logBle('讀取字串特徵失敗：${characteristic.uuid.str}，原因：$error', error: error, stackTrace: stackTrace);
  }
}

String _decodeUtf8String(List<int> data) {
  final trimmed = data.takeWhile((v) => v != 0).toList();
  if (trimmed.isEmpty) return '';
  return utf8.decode(trimmed, allowMalformed: true);
}
```

### ② 演算法 / 邏輯設計（Logic & Algorithm）

#### 特徵值通知啟用流程
**多次重試機制（Exponential Backoff）：**
1. 首次嘗試：立即啟用通知
2. 若失敗（GATT BUSY）：等待 400ms 後重試
3. 若再次失敗：等待 800ms 後重試
4. 若第三次失敗：等待 1200ms 後重試
5. 若全部失敗：降級至「僅讀取」模式

**去抖動與序列化機制：**
- 相鄰 CCCD 寫入間隔：180ms（防止 GATT 隊列溢出）
- 為每個設備維護單獨的通知訂閱清單
- 支援多設備（右腕、胸部）的並行監聽

#### 傳感器數據同步邏輯
**加速度 + 旋轉向量配對寫入：**
```
┌─────────────────────────────────────────────┐
│ 線性加速度隊列 → │
│                 ├→ 配對檢查 → CSV 寫入 → 檔案刷新
│ 旋轉向量隊列 ────→ │
└─────────────────────────────────────────────┘
```
- 只有兩個隊列都有數據時才執行寫入
- 防止單側數據堆積導致同步延遲

#### Game Rotation Vector 補償讀取邏輯
**時序：**
- 監聽通知有效期：啟用後持續監聽
- 超時檢測間隔：3 秒
- 觸發補償讀取條件：超過 3 秒未收到新數據

**補償讀取流程：**
1. 檢查上次更新時間戳
2. 若距離現在 > 3 秒，執行補償讀取
3. 確保特徵值支援讀取權限
4. 若讀取失敗且設備明確禁止讀取，停止計時器

```dart
// 來自 recorder_page.dart (L1285-1360)：通知訂閱統一入口
Future<bool> _initCharacteristic({
  required BluetoothCharacteristic characteristic,
  required void Function(String deviceId, List<int>) onData,
  bool listenToUpdates = true,
  bool readInitialValue = true,
  List<StreamSubscription<List<int>>>? targetSubscriptions,
  void Function(String message)? onErrorMessage,
}) async {
  bool shouldListen =
      listenToUpdates && (characteristic.properties.notify || characteristic.properties.indicate);

  if (shouldListen) {
    try {
      // ========== 檢查 CCCD 描述符 ==========
      if (characteristic.descriptors.isEmpty) {
        _logBle('特徵 ${characteristic.uuid.str} 未附帶描述符，改以直接開啟通知');
      } else {
        final hasCccd = characteristic.descriptors.any(
          (descriptor) => descriptor.uuid == _cccdUuid
        );
        if (!hasCccd) {
          _logBle('找不到 CCCD 描述符，改以直接開啟通知：${characteristic.uuid.str}');
        }
      }

      // ========== 啟用通知（含重試機制） ==========
      if (!characteristic.isNotifying) {
        shouldListen = await _enableNotificationWithRetry(characteristic);
      } else {
        _logBle('通知已開啟，略過重複設定：${characteristic.uuid.str}');
      }
    } catch (error, stackTrace) {
      shouldListen = false;
      _logBle('開啟通知失敗：${characteristic.uuid.str}，原因：$error',
          error: error, stackTrace: stackTrace);
    }
  }

  // ========== 訂閱通知流 ==========
  if (shouldListen) {
    final deviceId = characteristic.remoteId.str;
    final subscription = characteristic.lastValueStream.listen(
      (value) => onData(deviceId, value),
      onError: (error) {
        if (!mounted) return;
        if (onErrorMessage != null) {
          onErrorMessage('讀取 ${characteristic.uuid.str} 時發生錯誤：$error');
        } else {
          setState(() {
            _connectionMessage = '讀取 ${characteristic.uuid.str} 時發生錯誤：$error';
          });
        }
        _logBle('特徵通知流錯誤：${characteristic.uuid.str}，錯誤：$error', error: error);
      },
    );
    (targetSubscriptions ?? _notificationSubscriptions).add(subscription);
  }

  // ========== 初始讀取 ==========
  if (readInitialValue && characteristic.properties.read) {
    try {
      final value = await characteristic.read();
      if (value.isNotEmpty) {
        onData(characteristic.remoteId.str, value);
      }
      _logBle('已讀取初始值：${characteristic.uuid.str}，長度：${value.length}');
    } catch (error, stackTrace) {
      _logBle('初始讀取失敗：${characteristic.uuid.str}，原因：$error',
          error: error, stackTrace: stackTrace);
    }
  }

  return shouldListen;
}
```

**(2) 多次重試機制 `_enableNotificationWithRetry()`**：
```dart
// 來自 recorder_page.dart (L1212-1282)：多次重試機制
Future<bool> _enableNotificationWithRetry(
  BluetoothCharacteristic characteristic,
) async {
  const maxAttempts = 3;
  for (var attempt = 0; attempt < maxAttempts; attempt++) {
    var requireLongDelay = false;
    
    try {
      // 排入序列化佇列，避免同時對多個 CCCD 寫入
      await _bleOperationQueue.enqueue(
        () async {
          _logBle('排程啟用通知請求（第${attempt + 1}次）：${characteristic.uuid.str}');
          await characteristic.setNotifyValue(true);
        },
        spacing: const Duration(milliseconds: 180),
      );
      
      _logBle('成功啟用通知（第${attempt + 1}次嘗試）：${characteristic.uuid.str}');
      return true;
      
    } on FlutterBluePlusException catch (error, stackTrace) {
      final message = (error.errorString ?? '').toLowerCase();
      if (message.contains('busy')) {
        requireLongDelay = true;
        _logBle('偵測到底層回報忙碌，將延後下次重試：${characteristic.uuid.str}');
      }
      _logBle('開啟通知失敗（第${attempt + 1}次）：${characteristic.uuid.str} -> $error',
          error: error, stackTrace: stackTrace);
          
    } on PlatformException catch (error, stackTrace) {
      _logBle('開啟通知失敗（PlatformException，第${attempt + 1}次）：${characteristic.uuid.str} -> ${error.code}:${error.message}',
          error: error, stackTrace: stackTrace);
    } catch (error, stackTrace) {
      _logBle('開啟通知發生未預期例外（第${attempt + 1}次）：${characteristic.uuid.str} -> $error',
          error: error, stackTrace: stackTrace);
    }

    if (attempt < maxAttempts - 1) {
      final baseDelay = requireLongDelay ? 700 : 400;
      final delay = Duration(milliseconds: baseDelay * (attempt + 1));
      _logBle('通知啟用失敗，${delay.inMilliseconds}ms 後重試：${characteristic.uuid.str}');
      await Future.delayed(delay);
    }
  }

  _logBle('多次嘗試仍無法啟用通知，改採僅讀取模式：${characteristic.uuid.str}');
  return false;
}
```

```dart
// lib/services/audio_analyzer.dart (L150-L220)：FFT 頻譜分析
List<double> computeSpectrogram(List<int> audioSamples) {
  // 轉換為浮點數
  final floatSamples = audioSamples.map((s) => (s as double) / 32768.0).toList();
  
  // 應用 Hann 窗函數
  final windowed = _applyHannWindow(floatSamples);
  
  // 執行 FFT（使用 fft 套件）
  final fft = FFT(windowed.length);
  final spectrum = fft.realFFT(windowed);
  
  // 計算頻段能量（分為 256 個頻段）
  final energyBins = List<double>.filled(256, 0);
  for (int i = 0; i < spectrum.length; i++) {
    final binIdx = (i * 256 ~/ spectrum.length).clamp(0, 255);
    energyBins[binIdx] += spectrum[i].abs();
  }
  
  return energyBins;
}
```

```dart
// lib/services/audio_analysis_service.dart (L180-L200)：擊球偵測
Future<List<double>> detectGolfSwings(String videoPath) async {
  final audioSamples = await _extractAudioTrack(videoPath);
  final swingTimestamps = <double>[];
  
  // 以 512 樣本為滑動視窗，計算頻譜
  const windowSize = 512;
  const hop = 256;  // 50% 重疊
  const sampleRate = 44100;
  
  for (int i = 0; i < audioSamples.length - windowSize; i += hop) {
    final window = audioSamples.sublist(i, i + windowSize);
    final spectrum = computeSpectrogram(window);
    
    // 偵測 2-8 kHz 頻段（高頻衝擊聲）
    final freqBandIdx2k = (2000 * 256 ~/ (sampleRate / 2)).clamp(0, 255);
    final freqBandIdx8k = (8000 * 256 ~/ (sampleRate / 2)).clamp(0, 255);
    final bandEnergy = spectrum.sublist(freqBandIdx2k, freqBandIdx8k + 1)
        .reduce((a, b) => a + b);
    
    // 動態閾值：背景噪音的 3 倍
    if (bandEnergy > _backgroundNoise * 3.0) {
      final timestamp = (i + windowSize / 2) / sampleRate;
      swingTimestamps.add(timestamp);
      i += hop * 5;  // 跳過後續 5 個視窗避免重複偵測
    }
  }
  
  return swingTimestamps;
}
```

```dart
// lib/services/highlight_service.dart (L200-L320)：多模態融合與剪輯
Future<List<HighlightSegment>> generateHighlights(
  String csvPath,
  List<double> audioTimestamps,
) async {
  // 讀取 IMU CSV
  final lines = File(csvPath).readAsLinesSync();
  final segments = <HighlightSegment>[];
  
  const preSwingSeconds = 2.0;   // 擊球前 2 秒
  const postSwingSeconds = 3.0;  // 擊球後 3 秒
  const syncTolerance = 0.5;     // 時間同步容差 500ms
  
  for (final audioTs in audioTimestamps) {
    // 在 IMU 資料中尋找匹配的峰值
    double? imuPeakTs;
    double maxAccel = 0;
    
    for (final line in lines) {
      try {
        final parts = line.split(',');
        final timestampUs = int.parse(parts[0]);
        final timestamp = timestampUs / 1000000.0;
        
        // 檢查時間範圍 (audioTs ± 500ms)
        if ((timestamp - audioTs).abs() > syncTolerance) continue;
        
        // 計算加速度大小
        final ax = double.parse(parts[5]);
        final ay = double.parse(parts[6]);
        final az = double.parse(parts[7]);
        final accel = (ax * ax + ay * ay + az * az).sqrt();
        
        if (accel > maxAccel) {
          maxAccel = accel;
          imuPeakTs = timestamp;
        }
      } catch (e) {
        continue;
      }
    }
    
    // 雙重驗證成功
    if (imuPeakTs != null) {
      final startTime = (imuPeakTs - preSwingSeconds).clamp(0.0, double.infinity);
      final endTime = imuPeakTs + postSwingSeconds;
      
      segments.add(HighlightSegment(
        startTime: startTime,
        endTime: endTime,
        audioTimestamp: audioTs,
        imuTimestamp: imuPeakTs,
      ));
    }
  }
  
  return segments;
}
```

```dart
      return RecordingHistoryEntry.fromJson(jsonDecode(json));
    }).toList()
      ..sort((a, b) => b.recordedAt.compareTo(a.recordedAt));  // 依時間倒序
  }
  
  // 刪除錄影紀錄
  Future<void> deleteRecording(String recordingId) async {
    final prefs = await SharedPreferences.getInstance();
    final jsonList = prefs.getStringList(_key) ?? [];
    
    // 移除紀錄
    jsonList.removeWhere((json) {
      final entry = RecordingHistoryEntry.fromJson(jsonDecode(json));
      return entry.id == recordingId;
    });
    
    // 同步刪除檔案
    final entry = (await getRecordings())
        .firstWhereOrNull((e) => e.id == recordingId);
    if (entry != null) {
      await File(entry.videoPath).delete();
      await File(entry.imuCsvPath).delete();
    }
    
    // 更新持久化
    await prefs.setStringList(_key, jsonList);
  }
}
```
