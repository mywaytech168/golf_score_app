#### 1. User 模型 (`Models/User.cs`)
```csharp
public class User {
  public int Id { get; set; }
  public string Username { get; set; }      // 用戶名（登入名）
  public string Email { get; set; }         // 電子郵件
  public string PasswordHash { get; set; }  // 密碼雜湊（BCrypt）
  public string DisplayName { get; set; }   // 顯示名稱
  public string GoogleId { get; set; }      // Google OAuth ID
  public string AvatarUrl { get; set; }     // 用戶頭像
  public string Provider { get; set; }      // 登入方式: 'local' | 'google'
  public string Status { get; set; }        // 帳戶狀態: 'active' | 'inactive' | 'suspended'
  public DateTime CreatedAt { get; set; }
  public DateTime UpdatedAt { get; set; }
  public DateTime? LastLoginAt { get; set; }
}
```

#### 2. AuthService (`Services/AuthService.cs`)
核心功能：
- ✅ `RegisterAsync()` - 本地帳號註冊
- ✅ `LoginAsync()` - 本地帳號登入
- ✅ `GoogleLoginAsync()` - Google OAuth 登入/自動註冊
- ✅ `RefreshToken()` - 刷新 Token
- ✅ `GenerateTokens()` - JWT Token 生成
- ✅ `ChangePasswordAsync()` - 密碼變更

**密碼安全**：使用 BCrypt.Net 進行密碼雜湊和驗證

#### 3. AuthController (`Controllers/AuthController.cs`)
API 端點：

| 端點 | 方法 | 功能 | 驗證 |
|------|------|------|------|
| `/api/auth/register` | POST | 本地帳號註冊 | ❌ 無 |
| `/api/auth/login` | POST | 本地帳號登入 | ❌ 無 |
| `/api/auth/google-login` | POST | Google OAuth 登入 | ❌ 無 |
| `/api/auth/refresh-token` | POST | 刷新 Token | ⚠️ RefreshToken |
| `/api/auth/change-password` | POST | 密碼變更 | ✅ JWT |
| `/api/auth/logout` | POST | 登出 | ✅ JWT |

#### 4. JWT 身份驗證
```csharp
// Program.cs 配置
builder.Services.AddAuthentication(options => {
    options.DefaultAuthenticateScheme = "Bearer";
    options.DefaultChallengeScheme = "Bearer";
})
.AddJwtBearer(options => {
    options.TokenValidationParameters = new TokenValidationParameters {
        ValidateIssuerSigningKey = true,
        IssuerSigningKey = new SymmetricSecurityKey(Encoding.UTF8.GetBytes(jwtSecret)),
        ValidateIssuer = false,
        ValidateAudience = false,
        ValidateLifetime = true,
        ClockSkew = TimeSpan.Zero,
    };
});
```

## 🔧 Google OAuth 設置

### 1. Google Cloud 控制台配置
1. 前往 [Google Cloud Console](https://console.cloud.google.com/)
2. 建立新專案或選擇現有專案
3. 啟用 **Google+ API**
4. 建立 **OAuth 2.0 客戶端 ID**（Web 應用程式類型）
5. 新增授權的重定向 URI：
   - `http://localhost:5000/auth/google-callback`
   - `https://9eazqxfttdu2jv6j.atk.tw/auth/google-callback`

### 2. appsettings.json 配置
```json
"Google": {
  "ClientId": "YOUR_GOOGLE_CLIENT_ID.apps.googleusercontent.com",
  "ClientSecret": "YOUR_GOOGLE_CLIENT_SECRET",
  "RedirectUri": "http://localhost:5000/auth/google-callback"
}
```

### 3. Flutter 配置
**android/app/build.gradle**：
```gradle
dependencies {
    implementation 'com.google.android.gms:play-services-auth:20.0.0'
}
```

**iOS/Podfile**：
```ruby
pod 'GoogleSignIn', '~> 6.2'
```

**ios/Runner/Info.plist**：
```xml
<key>REVERSED_CLIENT_ID</key>
<string>com.googleusercontent.apps.YOUR_CLIENT_ID</string>
```

## 🔐 安全建議

### 產品環境配置

#### 1. 更改 JWT Secret
```json
// appsettings.Production.json
"Jwt": {
  "Secret": "USE_A_VERY_LONG_RANDOM_SECRET_AT_LEAST_32_CHARS_IN_PRODUCTION"
}
```

#### 2. 啟用 HTTPS
```csharp
app.UseHttpsRedirection();
```

#### 3. 設置安全 Header
```csharp
app.UseHsts(); // HTTP Strict Transport Security
```

#### 4. CORS 限制
```csharp
// 不要使用 AllowAnyOrigin() 在生產環境
options.AddPolicy("Production", policy =>
    policy.WithOrigins("https://yourdomain.com")
          .AllowAnyHeader()
          .AllowAnyMethod()
);
```
